Game Thoughts

This will be stuff that i thought while making the video game

Prisma Wings was really hard to make. For the main focus of this project I wanted to spend more time learning the new game engine Lumberyard and the Scripting language Lua. I had to make a presentable game though and it was a pretty good application of what I was learning in the Lumberyard Engine. The idea behind the game was to eventually make the multiplayer and have multiple players flying around and so it would be this experience of combining the different weapons. I tried to focus on making the weapons feel good to use and I think I did well.

This game was made for my Capstone project and one of the things our capstone was graded on was technical breath vs depth. I wanted to do a project that covered both of those topics and so I chose a relatively simple game idea and a new technology.
